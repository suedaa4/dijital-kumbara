# 💰 Digital Piggy Bank (Dijital Kumbara)

Bu proje, saf **JavaScript** kullanılarak geliştirilmiş, nesne (object) yönelimli programlama mantığını ve fonksiyonlar aracılığıyla veri güncellemeyi kavramayı amaçlayan temel bir algoritma alıştırmasıdır.

## 🚀 Özellikler
* Kumbaranın sahibini, toplam parasını ve hedefini bir nesne içinde tutma.
* Dinamik parametre kullanarak kumbaraya para ekleme (`addMoney`).
* `if / else` koşul mekanizması ile biriken paranın belirlenen hedefe ulaşıp ulaşmadığını otomatik hesaplama (`checkTarget`).

## 🛠️ Öğrenilen Temel Kavramlar
* **Objects (Nesneler):** İlişkili verileri bir arada gruplama.
* **Functions (Fonksiyonlar):** Belirli bir işi yapan kod blokları yazma ve parametre geçme.
* **Conditional Statements (`if/else`):** Koşullara göre farklı kod yollarını çalıştırma.

---

## 💻 Nasıl Çalıştırılır?

Projeyi kendi bilgisayarınızda test etmek için bir terminal açıp şu komutu yazmanız yeterlidir:
```bash
node app.js
```
